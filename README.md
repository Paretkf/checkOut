# tdd-ava-starter
A starter kit for learn TDD with AVA.

## Usage
```
$ yarn global add ava
```

```
$ git clone https://github.com/n9ti/tdd-ava-starter.git learn-tdd-with-ava
$ cd learn-tdd-with-ava
$ yarn
```

## Run it
```
$ yarn test
```

## Code coverage
```
$ yarn report
```

## Check JS Standard Style
```
$ yarn lint
```
